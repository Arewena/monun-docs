# 시작하기

import MavenVersion from "@site/src/components/MavenVersion"
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

## 개발 환경
    * JDK 17+
    * Kotlin
    * Paper 1.18+

## Tap의 기능 overview (Tap의 설명 中)
    - 개체 패킷
    - 가상 개체
    - 가상 발사체
    - 개체별 이벤트 리스너
    - YamlConfiguration을 이용한 문자열 템플릿
    - 추가적인 인벤토리 함수
    - GitHub를 통한 업데이트
    - Tick 기반 태스크 스케쥴러 (Ticker)
